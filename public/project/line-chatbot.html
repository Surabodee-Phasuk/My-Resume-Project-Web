Missing People

A team project developing a missing persons information system via LINE Chatbot to streamline reporting and tracking. Data is stored in a cloud database using AWS services to enhance efficiency, security, and searchability. เป็นโปรเจกต์ที่ทำเป็นทีมเป็นระบบข้อมูลผู้สูญหายผ่าน LINE Chatbot เพื่อให้การแจ้งเบาะแสและติดตามผู้สูญหายเป็นระบบมากขึ้น โดยจัดเก็บข้อมูลในฐานข้อมูลบนคลาวด์และใช้บริการ AWS เพื่อเพิ่มประสิทธิภาพ ความปลอดภัย และความสะดวกในการค้นหา

LINE Chatbot

Uses a LINE Chatbot to receive information via chat and upload it to the Cloud via AWS Services. The system is divided into reporting missing persons and reporting sightings/clues. เป็นการใช้ Line Chatbot ในการรับข้อมูลผ่านแชทเพื่อนำข้อมูลเข้าสู่ Cloud ผ่าน AWS Services โดยจะแบ่งเป็นการแจ้งข้อมูลคนหายและแจ้งการพบเบาะแส
